# Intervention Management Frontend

## Description

Application mobile cross-platform développée avec React Native permettant aux managers et aux techniciens de gérer et suivre les interventions terrain.

## Technologies utilisées

- React Native
- Expo
- TypeScript
- Axios
- AsyncStorage

## Fonctionnalités

### Manager
- Authentification
- Gestion des techniciens
- Création des interventions
- Affectation des interventions
- Suivi de l'avancement

### Technicien
- Consultation des interventions affectées
- Mise à jour du statut
- Rédaction des rapports
- Ajout de pièces jointes
